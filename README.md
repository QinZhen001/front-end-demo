#### vertical-center-picture
大小不固定的图片的垂直居中(3种实现方式)

1. **将要显示的图片与一张透明且高度100%，宽度1像素的图片都设置vertical-align:middle，使之对齐。**
2. 在图片的父元素上添加display: table-cell vertical-align: middle;
3. 使用flex布局 (**推荐使用**)
